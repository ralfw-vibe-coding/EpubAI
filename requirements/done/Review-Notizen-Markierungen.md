# Abnahme: Notizen & Markierungen


## Markieren

Im Reader einen Textabschnitt auswählen (antippen und Auswahl ziehen). Es sollte eine kleine Leiste mit „Markieren" erscheinen. Nach dem Antippen sollte der ausgewählte Text farbig hinterlegt sein (Akzentfarbe, leicht transparent).

## Markieren am Zeilenrand

Mit der Maus über den Text fahren — der Mauszeiger sollte sich (außer auf der Randfläche neben dem Text) durchgehend als Textcursor zeigen, auch unmittelbar vor dem ersten bzw. nach dem letzten Buchstaben einer Zeile. Eine Markierung sollte dort genauso beginnen können wie in der Textmitte.

## Notiz hinzufügen

Eine gerade erstellte Markierung antippen. Es sollte sich unten ein Editor mit Textfeld öffnen. Text eingeben und „Speichern" antippen. Die Notiz sollte danach in der Notizen-Liste sichtbar sein.

## Notiz bearbeiten

Eine Markierung mit bestehender Notiz erneut antippen, den Text ändern und speichern. Die Änderung sollte übernommen werden.

## Markierung löschen

Eine Markierung über den Notiz-Editor („Löschen") oder direkt in der Notizen-Liste (Papierkorb-Symbol) entfernen. Die farbige Hervorhebung im Text sollte danach verschwinden.

## Notizen-Liste

Über das neue Symbol (Textmarker-Icon) im Reader-Header öffnet sich eine Liste aller Markierungen/Notizen zu diesem Buch. Ein Eintrag antippen sollte an die entsprechende Stelle im Buch springen.

## Persistenz beim erneuten Öffnen

Buch schließen und erneut öffnen. Alle zuvor erstellten Markierungen sollten wieder sichtbar sein — sie werden lokal zwischengespeichert, dafür ist keine Internetverbindung nötig.

## Sync beim App-Start

Beim Start der App (Bibliotheksansicht) sollten alle Markierungen im Hintergrund mit dem Server abgeglichen werden. Am einfachsten so zu prüfen: eine Markierung anlegen, App bzw. Tab schließen, neu öffnen und nachsehen, ob sie noch da ist.

## Offline-Fall (optional)

Falls gewünscht: eine Markierung anlegen, während keine Verbindung zum Server besteht (z. B. Backend kurz stoppen). Es sollte eine Fehlermeldung erscheinen („Markierung konnte nicht gespeichert werden — keine Verbindung."), keine fehlerhaft angelegte Markierung.
